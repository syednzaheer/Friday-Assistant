## Friday – Python Desktop Voice Assistant

Friday is a lightweight desktop voice assistant built using Python.
It performs common automation and utility tasks locally through voice commands.
The project focuses on practical Python usage, automation logic, and debugging — without relying on external APIs or cloud services.

## Features

Voice command recognition
Local application and website launching
Music playback from local system
Text-to-speech responses
Improved command handling and error control
Runs entirely offline

## Technologies Used

Python
SpeechRecognition
pyttsx3
gTTS
OS & automation libraries

## How to Run

Make sure Python is installed
Install required dependencies

## Run the Main Script:

python friday.py

## Project Notes

This project was built incrementally by starting from a basic voice assistant structure and improving it through experimentation, debugging, and feature extensions.
The focus was on:
Understanding voice input pipelines
Handling edge cases gracefully
Writing readable and maintainable Python code
